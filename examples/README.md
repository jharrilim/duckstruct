# Duckstruct examples

## basic

A minimal Duckstruct project: a simple object type and string concatenation.

**Run from repo root:**

```bash
# Compile to JavaScript (output in examples/basic/output/js/)
duckstruct compile examples/basic
```

Or from inside the example directory:

```bash
cd examples/basic
duckstruct compile .
```

**Scaffold a new project like this:**

```bash
duckstruct init my-project
cd my-project
duckstruct compile .
```
