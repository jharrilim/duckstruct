# Duckstruct User Guide

Duckstruct is a programming language focused on structural typing through
"duck constraints". This documentation is for writing programs in Duckstruct,
using the CLI, and understanding language behavior.

## Start here

- New to Duckstruct? Read [Quickstart](./quickstart)
- Learning syntax? Read [Syntax Basics](./syntax-basics)
- Working with data? Read [Core Types and Collections](./core-types)
- Building reusable code? Read [Functions and Traits](./functions-and-traits)
- Splitting code across files? Read [Modules and Imports](./modules-and-imports)
- Using the tooling? Read [CLI Reference](./cli)
